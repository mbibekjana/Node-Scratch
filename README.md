## installation

First copy the project.
private
Use git clone https://mbibekjana@bitbucket.org/mbibekjana/nodescratch.git.
public
git clone https://github.com/mbibekjana/Node-Scratch.git

Then go to project folder.

Run npm install.
sequelize init
